## 🆘 **TROUBLESHOOTING**

### Common Issues

**Sub-agent not responding?**
```bash
subagents action=list
subagents action=steer target=[id] message="[guidance]"
```

**Memory search slow?**
```bash
# Check cache config
cat config/performance.json | grep -A5 caching
# Cache auto-cleans every 6 hours
```

**Backup failing?**
```bash
# Check disk space
df -h

# Check permissions
ls -la backup/
```

**CLI not working?**
```bash
# Make executable
chmod +x cli/clawd.sh

# Test
./cli/clawd.sh help
```
