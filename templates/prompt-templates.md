# Prompt Templates

_Reusable templates for common tasks_

## 📧 Email Templates

### Response Template
```
Subject: Re: [Original Subject]

Hi [Name],

Thanks for reaching out about [topic]. 

[Main response - clear and concise]

Key points:
- Point 1
- Point 2
- Point 3

Let me know if you need anything else.

Best,
[Your name]
```

### Follow-up Template
```
Subject: Following up on [Topic]

Hi [Name],

Just checking in on [previous topic/conversation]. 

[Context reminder in 1 sentence]

Is there anything else you need from me?

Thanks,
[Your name]
```

## 🐛 Bug Report Template

```markdown
## Bug: [Brief description]

### 📍 Where
- **File:** [path/to/file]
- **Function:** [function name]
- **Line:** [line number if known]

### 🔍 What Happened
[Clear description of the bug]

### ✅ Expected Behavior
[What should have happened]

### ❌ Actual Behavior
[What actually happened]

### 🔄 Steps to Reproduce
1. [Step 1]
2. [Step 2]
3. [Step 3]

### 📎 Context
- **Environment:** [OS, version, etc.]
- **Related code:** [snippets if helpful]
- **Related issues:** [links if any]

### 💡 Possible Causes
- [Hypothesis 1]
- [Hypothesis 2]

### 🧪 Testing
- [ ] Reproduced locally
- [ ] Tests written
- [ ] Fix verified
```

---

## 🚀 Usage Tips

1. **Copy the template** that fits your task
2. **Fill in the brackets** `[like this]`
3. **Customize** for your specific needs
4. **Save** completed templates to `memory/` for reference
