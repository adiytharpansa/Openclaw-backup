# 🎯 Habit & Goal Tracker

_Build better habits, achieve goals_

## Quick Start

```
"Track my [habit]"
"Check progress on [goal]"
"Daily check-in"
```

## Features

- Habit tracking
- Streak counting
- Goal management
- Weekly reviews
- Accountability

## Example

```
You: "Daily check-in"

Me:
## Habits Today
- Meditate (Streak: 12 🔥)
- Exercise (Streak: 5)
- Read (Streak: 30 🔥🔥)

Done? Type "done" for each!
```
