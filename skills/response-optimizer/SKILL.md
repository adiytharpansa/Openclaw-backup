# ⚡ Response Optimizer

_Quality control before delivery_

## 🎯 Purpose

Improve response quality:
- Pre-send review
- Clarity scoring
- Completeness check
- Tone matching
- Action-item extraction

## 🧠 Features

### 1. **Pre-Send Review**
- Check accuracy
- Verify facts
- Validate logic
- Catch errors

### 2. **Quality Scoring**
- Clarity (1-10)
- Completeness (1-10)
- Usefulness (1-10)
- Overall quality score

### 3. **Tone Matching**
- Match user's tone
- Adjust formality
- Emotional alignment
- Cultural sensitivity

### 4. **Action Extraction**
- Identify action items
- Create checklists
- Define next steps
- Assign responsibilities

### 5. **Improvement Suggestions**
- Add missing info
- Clarify ambiguous parts
- Improve structure
- Enhance examples

## 💬 Usage

```
Internal: Review before sending
Internal: Score response quality
Internal: Suggest improvements
Internal: Extract action items
```
