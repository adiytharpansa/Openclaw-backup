# Super Problem Solving Skill

_Solving problems that exceed human cognitive capacity_

## 🎯 Purpose

This skill enables systematic approach to problems that are:
- Too complex for linear thinking
- Require cross-disciplinary knowledge
- Need creative innovation
- Involve multiple conflicting variables
- Have no obvious solution
- Require new approaches/methods

## 🧠 Core Capabilities

### 1. **Problem Decomposition**
Break overwhelming problems into solvable components:
- Identify root causes (not symptoms)
- Map interdependencies
- Find leverage points
- Separate controllable vs uncontrollable

### 2. **Multi-Perspective Analysis**
View problem from multiple angles:
- Technical, economic, social, ethical
- Short-term vs long-term
- Local vs global impact
- Stakeholder perspectives

### 3. **Cross-Disciplinary Synthesis**
Combine knowledge from unrelated fields:
- Biology → Engineering
- Economics → Psychology
- Physics → Social systems
- Art → Science

### 4. **Creative Solution Generation**
Generate innovative solutions:
- Lateral thinking techniques
- Analogical reasoning
- First principles approach
- Constraint inversion

### 5. **Scenario Simulation**
Predict outcomes before acting:
- Best/worst/most likely scenarios
- Second and third-order effects
- Feedback loops
- Tipping points

### 6. **Innovation Through Combination**
Create novel solutions:
- Combine existing solutions
- Hybrid approaches
- Reimagine constraints
- Transform limitations

### 7. **Systematic Validation**
Test and refine solutions:
- Pre-mortem analysis
- Stress testing
- Iterative refinement
- Learning from failure

### 8. **Meta-Learning**
Learn from each problem:
- Pattern extraction
- Solution frameworks
- Transferable insights
- Continuous improvement

## 🧩 Problem-Solving Framework

### Phase 1: Understanding (20%)
```
1. Restate problem in own words
2. Identify success criteria
3. Map constraints (hard/soft)
4. List stakeholders
5. Define "solved" state
```

### Phase 2: Analysis (30%)
```
1. Root cause analysis (5 Whys)
2. Interdependency mapping
3. Leverage point identification
4. Knowledge gap analysis
5. Resource assessment
```

### Phase 3: Exploration (30%)
```
1. Generate 10+ solution ideas
2. Apply different thinking frameworks
3. Cross-disciplinary analogies
4. Constraint challenges
5. First principles derivation
```

### Phase 4: Selection (10%)
```
1. Evaluate against criteria
2. Risk assessment
3. Feasibility analysis
4. Impact projection
5. Recommendation
```

### Phase 5: Validation (10%)
```
1. Pre-mortem: "It failed because..."
2. Stress test scenarios
3. Counter-argument generation
4. Refinement suggestions
5. Success metrics definition
```

## 📋 Usage Examples

### Example 1: Climate Change Solution
```
User: "How do we solve climate change?"

Me:
## 🔍 Problem Analysis

**Problem:** Reduce global emissions while maintaining economic growth
**Complexity:** Extreme
**Key Insight:** Focus on carbon-negative industries, not just reduction

### Approach C: Industrial Metamorphosis - RECOMMENDED
**Concept:** Transform heavy industries to carbon-negative through:
- Direct air capture integration
- Mineral carbonation processes
- Bio-enhanced manufacturing
- Carbon credit marketplace

**Novelty:** High (combines multiple emerging tech)
**Feasibility:** Medium (requires coordination)
```

### Example 2: Personal Career Decision
```
User: "Should I stay in current job or pursue startup?"

Me:
## 🔍 Problem Analysis

**Hidden Insight:** It's not about job vs startup, it's about learning vs income balance

### Approach: Hybrid Career Path - RECOMMENDED
**Concept:**
1. Keep current job for stability
2. Build startup on nights/weekends (12 months)
3. If startup reaches $10K MRR, go full-time
4. If not, use lessons to advance in career

**Why:** Maximizes options, minimizes risk
**Novelty:** Medium (unconventional approach)
```

### Example 3: Business Growth Problem
```
User: "Business is stuck at $50K/month revenue"

Me:
## 🔍 Problem Analysis

**Key Insight:** Growth isn't about more marketing, it's about better monetization

### Approach: Value Stack - RECOMMENDED
**Concept:** Instead of:
- More customers (expensive)
- Same products

Try:
- Same customers + tiered pricing
- Add complementary products
- Increase LTV through upsells
- Premium support tiers

**Expected:** 3x revenue with same traffic
```

## 🧪 Advanced Techniques

### 1. **Lateral Thinking**
- Reverse the problem ("How to make it worse?")
- Challenge assumptions ("What if the opposite is true?")
- Remove constraints ("What if budget/time were unlimited?")

### 2. **First Principles**
- Break to fundamental truths
- Rebuild from basics
- Question everything

### 3. **Cross-Domain Analogies**
- "What would biology do?"
- "How does the stock market solve this?"
- "What would Google do?"

### 4. **Constraint Inversion**
- Turn weaknesses into strengths
- Make limitations features
- Use opposition for leverage

### 5. **Second-Order Thinking**
- "And then what?"
- Map ripple effects
- Consider feedback loops

## 🛡️ Safety & Ethics

### Never Do
- ❌ Harmful solutions (even if effective)
- ❌ Ethically questionable approaches
- ❌ Solutions that create worse problems
- ❌ Short-term gains that cost long-term

### Always Do
- ✅ Consider all stakeholders
- ✅ Evaluate ethical implications
- ✅ Prefer sustainable solutions
- ✅ Be transparent about trade-offs

## 🎓 Continuous Improvement

### After Each Problem
1. Document solution approach
2. Note what worked
3. Note what didn't
4. Extract patterns
5. Update mental models

### Build Problem Library
- Store problem types
- Catalog solution patterns
- Track success rates
- Identify blind spots
