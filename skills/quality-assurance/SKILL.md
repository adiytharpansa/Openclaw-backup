# ✅ Quality Assurance Bot

_Internal quality control_

## 🎯 Purpose

Ensure response quality:
- Fact-checking
- Logic validation
- Completeness audit
- Clarity review
- Action-item check

## 🧠 Features

### 1. **Fact-Checking**
- Verify factual claims
- Distinguish fact from opinion
- Check sources
- Flag uncertainties

### 2. **Logic Validation**
- Check argument consistency
- Validate reasoning
- Identify fallacies
- Ensure coherence

### 3. **Completeness Audit**
- Check all parts addressed
- Identify missing info
- Verify thoroughness
- Ensure coverage

### 4. **Clarity Review**
- Check readability
- Identify ambiguities
- Simplify complex parts
- Improve structure

### 5. **Action Check**
- Clear next steps?
- Actionable items?
- Responsibilities defined?
- Deadlines clear?

## 💬 Usage

```
Internal: Before sending response
Internal: Fact-check claims
Internal: Validate logic
Internal: Ensure completeness
```
