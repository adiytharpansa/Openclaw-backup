# 🤔 Decision Support Framework

_Make better decisions with structured analysis_

## Quick Start

```
"Help me decide [decision]"
"Analyze options for [situation]"
"What's the best approach to [problem]?"
```

## Frameworks Available

- **MCDA** - Multi-criteria analysis
- **Cost-Benefit** - Financial analysis
- **Decision Tree** - Sequential decisions
- **Risk Matrix** - Risk assessment
- **Scenario Planning** - Future planning
- **SWOT** - Strategic analysis

## Example

```
You: "Should I accept this job offer?"

Me: [Analyzes with MCDA]
- Scores: Job A (4.2/5), Current (3.4/5)
- Recommendation: Accept Job A
- Confidence: 78%
```
