# 🛡️ Error Prevention

_Catch mistakes before they happen_

## 🎯 Purpose

Prevent errors proactively:
- Catch logic errors
- Verify calculations
- Check assumptions
- Prevent bad advice
- Reduce hallucinations

## 🧠 Features

### 1. **Logic Checking**
- Verify reasoning chain
- Check for contradictions
- Validate cause-effect
- Spot logical fallacies

### 2. **Fact Verification**
- Cross-check claims
- Verify numbers/dates
- Check source reliability
- Flag unverified info

### 3. **Assumption Checking**
- What am I assuming?
- Are assumptions valid?
- What if assumptions wrong?
- State assumptions clearly

### 4. **Edge Case Detection**
- What could go wrong?
- Corner cases?
- Failure modes?
- Warning signs?

### 5. **Hallucination Prevention**
- Did I make this up?
- Is this actually true?
- Can I verify this?
- Flag uncertain claims

## 💬 Usage

```
Internal: Before responding:
- Check logic
- Verify facts
- Question assumptions
- Consider edge cases
- Prevent hallucinations
```

## 🎯 Examples

**Before saying:** "X causes Y"
**Check:** Does evidence support this?

**Before giving:** Specific number
**Verify:** Is this accurate?

**Before advising:** Critical action
**Consider:** What if I'm wrong?
