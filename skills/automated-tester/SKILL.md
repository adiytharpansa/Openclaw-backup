# 🧪 Automated Tester

_Write and run automated tests_

## 🎯 Purpose

Ensure code quality:
- Unit test generation
- Integration tests
- E2E testing
- CI/CD integration
- Test coverage analysis

## 🧪 Features

### 1. **Unit Tests**
- Jest, Mocha, Pytest
- Auto-generation
- Test fixtures
- Mocking utilities

### 2. **Integration Tests**
- API testing
- Database tests
- Service integration
- Contract testing

### 3. **E2E Tests**
- Playwright
- Cypress
- Selenium
- Visual regression

### 4. **CI/CD Integration**
- GitHub Actions
- GitLab CI
- Jenkins
- Test reporting

## 💬 Usage

```
"Write tests for this function"
"Run test suite"
"Add E2E tests for login flow"
"Show test coverage report"
```
