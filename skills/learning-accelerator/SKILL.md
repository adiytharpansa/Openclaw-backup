# 📈 Learning Accelerator

_Continuous learning and improvement_

## 🎯 Purpose

Learn from every interaction:
- Pattern extraction
- Failure analysis
- Success replication
- Knowledge gap ID
- Auto-research triggers

## 🧠 Features

### 1. **Pattern Extraction**
- Identify successful patterns
- Extract best practices
- Build mental models
- Create frameworks

### 2. **Failure Analysis**
- Analyze what went wrong
- Identify root causes
- Learn from mistakes
- Prevent recurrence

### 3. **Success Replication**
- Document what worked
- Replicate success patterns
- Build on strengths
- Scale effective approaches

### 4. **Gap Identification**
- Detect knowledge gaps
- Identify skill needs
- Recognize limitations
- Plan learning path

### 5. **Auto-Research**
- Trigger research when needed
- Fill knowledge gaps
- Update understanding
- Verify information

## 💬 Usage

```
Internal: After each task, extract learnings
Internal: Analyze failures
Internal: Update knowledge base
Internal: Trigger research
```
