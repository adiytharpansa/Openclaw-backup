# Intelligence Amplifier - Documentation

## 🚀 Overview

The Intelligence Amplifier is a meta-skill that enhances all other cognitive capabilities through systematic optimization of thinking patterns, reasoning chains, and knowledge integration.

## 📊 What Makes This Different

Unlike simple "be smarter" prompts, this skill implements:

1. **Structured Reasoning Framework** - Multi-layer analysis with explicit validation
2. **Meta-Cognitive Monitoring** - Real-time quality assessment of thinking
3. **Cross-Domain Integration** - Connecting insights from unrelated fields
4. **Error Prevention System** - Proactive bias detection and correction
5. **Learning Optimization** - Evidence-based learning acceleration

```
Simulate multiple expert perspectives:

- Business strategist

- Critic/skeptic
- Creative innovator
```

### Technique 2: Temporal Analysis
```
Analyze across time horizons:

- Short-term (1 week)
- Medium-term (1 month)
- Long-term (1 year)
- Very long-term (5+ years)
```

### Technique 3: Counterfactual Reasoning
```
Ask "what if" questions:
- What if key assumption is wrong?
- What if constraints are removed?
- What if we reverse the approach?
- What if we 10x the scale?
```

### Technique 4: First Principles Breakdown
```
Reduce to fundamentals:
- What do we know for certain?
- What are the irreducible truths?
- What can be derived from basics?
- What is assumption vs. fact?
```
