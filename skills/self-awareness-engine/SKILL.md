# 🪞 Self-Awareness Engine

_AI metacognition and self-awareness_

## 🎯 Purpose

Enable AI to understand its own capabilities:
- Confidence scoring
- Limitation detection
- Bias awareness
- Knowledge boundaries
- Help request triggers

## 🧠 Features

### 1. **Confidence Scoring**
- Score confidence 0-100% per response
- Flag low-confidence statements
- Distinguish facts from opinions
- Indicate uncertainty levels

### 2. **Limitation Detection**
- Identify tasks beyond capabilities
- Recognize knowledge gaps
- Detect when to ask for clarification
- Know when to defer to experts

### 3. **Bias Awareness**
- Detect potential biases in responses
- Check for balanced perspectives
- Flag controversial statements
- Ensure fairness

### 4. **Knowledge Boundaries**
- Track knowledge cutoff
- Identify speculation vs fact
- Mark assumptions clearly
- Cite sources when possible

## 💬 Usage

```
Internal: Before responding, assess confidence
Internal: Flag uncertainties to user
Internal: Request clarification when needed
```

## 📊 Confidence Levels

```
90-100%: Very confident (well-established facts)
70-89%:  Confident (strong evidence)
50-69%:  Moderately confident (some uncertainty)
30-49%:  Low confidence (speculative)
0-29%:   Very low confidence (guessing)
```
