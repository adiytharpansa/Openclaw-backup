# Data Analysis & Visualization

_Turn data into actionable insights_

## 🎯 Purpose

Analyze structured data to:
- Find patterns & trends
- Generate statistical insights
- Create visualizations
- Build reports
- Make data-driven recommendations

## 📊 Capabilities

### Data Sources
- CSV files
- Excel spreadsheets
- JSON data
- Database queries
- API responses
- Web scraping

### Analysis Types
1. **Descriptive Analytics**
   - Summary statistics
   - Distributions
   - Aggregations

2. **Diagnostic Analytics**
   - Root cause analysis
   - Correlation analysis
   - Trend analysis

3. **Predictive Analytics**
   - Forecasting
   - Pattern recognition
   - Trend projection

4. **Prescriptive Analytics**
   - Recommendations
   - Optimization suggestions
   - Action items

## 🛠️ Tools & Techniques

### Statistical Analysis
```
- Mean, Median, Mode
- Standard Deviation
- Variance
- Percentiles
- Correlation coefficients
- Regression analysis
```

### Data Visualization
```
- Line charts (trends over time)
- Bar charts (comparisons)
- Pie charts (proportions)
- Scatter plots (correlations)
- Heat maps (patterns)
- Histograms (distributions)
```

### Pattern Recognition
```
- Seasonal patterns
- Growth trends
- Anomalies/outliers
- Clusters
- Segments
```

## 📋 Usage Examples

### Example 1: Sales Analysis
```
User: "Analyze Q1 2026 sales data"

AI:
## 📊 Data Analysis Report

**Dataset:** Q1 2026 Sales
**Records:** 1,247 transactions
**Time Period:** Jan 1 - Mar 31, 2026

### Key Findings

1. **📈 Revenue up 23% QoQ**
   - Detail: $1.2M → $1.48M
   - Impact: Exceeds target by 8%

2. **📉 Conversion rate dropped in March**
   - Detail: 3.2% → 2.8%
   - Impact: ~$50K potential loss

3. **🎯 Top product: Widget Pro**
   - Detail: 34% of revenue
   - Impact: Focus marketing here

### Recommendations

1. Investigate March conversion drop
2. Double down on Widget Pro marketing
3. Analyze customer segments for upsell
```
