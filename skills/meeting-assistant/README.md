# 📋 Meeting Assistant

_Automated meeting documentation_

## Quick Start

```
"Join my [time] meeting"
"Take notes for [meeting]"
"Extract action items"
```

## Features

- Live transcription
- Note taking
- Action item extraction
- Follow-up emails
- Decision logging

## Example

```
You: "Notes from yesterday's meeting"

Me:
- Decisions: 4 logged
- Actions: 7 created
- Summary: [sent to attendees]
- Your tasks: [added to todo]
```
