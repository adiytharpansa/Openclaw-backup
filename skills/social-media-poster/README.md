# 📱 Social Media Auto-Poster

Schedule and post to Twitter, LinkedIn, Instagram automatically.

## Quick Start

```
"Post to Twitter: [content]"
"Schedule for Monday 9 AM"
"Show this week's analytics"
```

## Features

✅ Multi-platform posting
✅ Content scheduling
✅ Content calendar
✅ Analytics tracking
✅ Auto-formatting
