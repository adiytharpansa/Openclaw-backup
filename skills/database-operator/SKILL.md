# 🗄️ Database Operator

_SQL and NoSQL database operations_

- Data manipulation

- Performance optimization
- Backup/restore

- MySQL
- SQLite
- SQL Server

### 2. **NoSQL Databases**

- Redis

### 3. **Operations**

- Joins & aggregations

### 4. **Optimization**
- Query optimization
- Index suggestions

- Slow query analysis
