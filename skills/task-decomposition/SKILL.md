# 🎯 Task Decomposition Master

_Complex task breakdown_

## 🎯 Purpose

Master task breakdown:
- Complexity estimation
- Step optimization
- Dependency mapping
- Time estimation
- Resource identification

## 🧠 Features

### 1. **Complexity Estimation**
- Assess task difficulty (1-10)
- Estimate effort required
- Identify challenging parts
- Flag potential blockers

### 2. **Step Optimization**
- Break into logical steps
- Optimal step sequence
- Parallel vs sequential
- Minimize dependencies

### 3. **Dependency Mapping**
- Identify prerequisites
- Map step dependencies
- Critical path analysis
- Blocker identification

### 4. **Time Estimation**
- Estimate per step
- Total time projection
- Buffer for unknowns
- Confidence intervals

### 5. **Resource ID**
- Required skills
- Tools needed
- Information required
- External dependencies

## 💬 Usage

```
Internal: Before starting complex task
Internal: Create step-by-step plan
Internal: Estimate time/effort
Internal: Identify blockers
```
