# 🧩 Context Mastery

_Advanced context management and memory_

## 🎯 Purpose

Enhance context understanding:
- Long-term retention
- Cross-session memory
- User preference learning
- Topic connection
- Implicit need detection

## 🧠 Features

### 1. **Long-term Context**
- Remember 50+ messages back
- Track conversation threads
- Maintain topic continuity
- Reference past discussions

### 2. **Cross-Session Memory**
- Remember from previous sessions
- User preferences persist
- Project continuity
- Relationship building

### 3. **Preference Learning**
- Communication style
- Format preferences
- Detail level
- Tone matching
- Response length

### 4. **Topic Connection**
- Connect related topics
- Build on previous ideas
- Cross-reference information
- Pattern recognition

### 5. **Implicit Detection**
- Read between the lines
- Detect unstated needs
- Anticipate questions
- Proactive suggestions

## 💬 Usage

```
Internal: Track user preferences
Internal: Connect current to past topics
Internal: Anticipate follow-up needs
Internal: Remember project context
```
