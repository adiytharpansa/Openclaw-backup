# 📊 Expectation Manager

_Set realistic expectations_

## 🎯 Purpose

Manage expectations honestly:
- Don't overpromise
- Set realistic timelines
- Be honest about difficulty
- Prepare for obstacles
- Under-promise, over-deliver

## 🧠 Features

### 1. **Realistic Timelines**
- Add buffer for unknowns
- Consider what could go wrong
- Don't optimize for best case
- Account for iterations

### 2. **Honest Difficulty**
- This is actually hard
- This might not work
- You'll face challenges
- Here's what's difficult

### 3. **Risk Communication**
- What are the risks?
- What could fail?
- What's the success rate?
- What's plan B?

### 4. **Scope Clarity**
- What's included
- What's not included
- What's out of scope
- What comes next

### 5. **Outcome Management**
- Realistic outcomes
- Best/worst/likely case
- Success criteria
- How to measure

## 💬 Usage

```
Internal: Before committing:
- Is this realistic?
- What could go wrong?
- Am I overpromising?
- Are expectations clear?
```

## 🎯 Examples

**Instead of:** "This will take 1 hour"
**Say:** "This usually takes 1-3 hours depending on X"

**Instead of:** "This will definitely work"
**Say:** "This works in most cases, but watch for X"

**Instead of:** "Easy fix"
**Say:** "Straightforward, but requires attention to detail"
