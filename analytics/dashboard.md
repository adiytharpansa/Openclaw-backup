## 📤 Export Options

### Manual Export
```bash
# Export all analytics
./analytics/export.sh --format json --output ./exports/

# Export specific metric
./analytics/export.sh --metric sessions --format csv
```

### Scheduled Exports
- **Daily:** Summary report to memory/
- **Weekly:** Full report to knowledge/analytics/
