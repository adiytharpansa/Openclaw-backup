## 🆘 **TROUBLESHOOTING**

### Service not starting?
```bash
# Check logs
docker-compose logs -f <service>

# Restart
docker-compose restart <service>

# Check health
curl localhost:9090 (Prometheus)
curl localhost:3000 (Grafana)
```

### Database connection failed?
```bash
# Check PostgreSQL
docker exec clawdb-postgres pg_isready -U clawd

# Check Redis
docker exec clawdb-redis redis-cli ping

# Check MongoDB
docker exec clawdb-mongodb mongosh --eval "db.adminCommand('ping')"
```

### API returning 500?
```bash
# Check API logs
docker-compose logs -f api-gateway

# Check health
curl localhost:8000/health

# Check dependencies
curl localhost:8000/health/db
curl localhost:8000/health/redis
curl localhost:8000/health/elasticsearch
```
